# cs4alhaider's Homebrew Tap

A personal [Homebrew](https://brew.sh) tap. Add it once, then install any of the tools below:

```sh
brew tap cs4alhaider/tap
```

## claudee

[claudee](https://github.com/cs4alhaider/claudee) — an interactive terminal launcher for Claude Code: a fast arrow-key wizard for git worktrees, resuming past conversations, and launch options.

```sh
brew install cs4alhaider/tap/claudee
```

## EnvHub

[EnvHub](https://github.com/cs4alhaider/EnvHub) — a native macOS app and CLI for managing every `.env` file on your machine.

```sh
# The app (installs the `envhub` CLI too)
brew install --cask cs4alhaider/tap/envhub-app

# Just the CLI
brew install cs4alhaider/tap/envhub
```
