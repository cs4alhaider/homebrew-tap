# EnvHub Homebrew Tap

Install [EnvHub](https://github.com/cs4alhaider/EnvHub) — a native macOS app and CLI for
managing every `.env` file on your machine.

```sh
# The app (installs the `envhub` CLI too)
brew install --cask cs4alhaider/tap/envhub-app

# Just the CLI
brew install cs4alhaider/tap/envhub
```
